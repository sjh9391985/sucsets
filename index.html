프로젝트 요약

  • # 반응형 웹 개발
  • - SPA 아키텍처 구조의 주식 추천 및 사전 웹 개발
  • - 키움증권 Open API를 사용한 주식 데이터 분석
  • - 부트스트랩을 커스텀을 통한 UI/UX 웹 개발 및 퍼블리싱

    - 주식 데이터 Open API의 사용법 숙지 어려움 및 대용량 데이터 이슈로 인한 데이터 가공이슈

  • - 적절한 데이터 분석 계산식을 세운 후 필터링을 통해 데이터 저장으로 해결
  • #2

  • #3

    - 웹 서버 배포 및 자동화에 있어 배포 서버 구축의 어려움

  • Project 소스코드: Sucset

프로젝트 개요 및 역할

울산대학교 캡스톤 디자인 프로젝트로 저조한 은행 금리와 안정성 있는 투자 수익률을 얻고자 제작

2명의 프로젝트 팀원으로 프론트 엔드와 백엔드 및 데이터 분석의 역할을 담당하여 제작

Html, Css, JavaScript 를 통한 프론트 UI 웹 개발 및 HIGHCHART 를 사용한 데이터 분석 시각화 구현

사용자가 주식 사전 검색을 하도록 검색 기능을 구현 및 추천 주식 종목에 따른 데이터 시각화 필터링 구현


[기술 스택]
  • 🔨 Front End : HTML5, CSS, JavaScript, HIGHCHART
  • 🔨 Back End : Django
  • 🔨 DB : SQLite3
  • 🔨 Data Analysis : Pandas, Matplotlib, NumPy
